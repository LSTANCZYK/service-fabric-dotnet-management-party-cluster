  • what's a party cluster?

    Party clusters are open, public Service Fabric clusters hosted on Azure and run by the Service Fabric team where anyone can deploy applications and learn about the platform. For free.

  • how does it work?

    Just click to join a cluster and we'll send you information on how to connect and deploy your applications. No subscriptions or sign-ups required.

    Each party cluster is the same and has a time limit. It runs for a few hours, then it's automatically taken down and a fresh new one comes up. At that point, all your applications will be removed and you'll have to connect to a new cluster to keep partying. You can only connect to one party cluster at a time, but you can come back as many times as you like.

  • what are the rules?

    You can deploy any application you like to learn about the platform - we suggest starting with one of our sample applications. But remember, these clusters are public and shared with others, so everyone can see your applications. Please see our Terms of Use for all the details.

  • cool, does this run on Service Fabric too?

    Of course! The Party Clusters and this web page are managed by a Service Fabric application which is open source and available on GitHub.
